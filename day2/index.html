Task 1:
1. var and const both initializes variables.
var has function scope and variables initialized with it can be reassigned.
const has block scope and variables initialized with it cannot be reassigned.
2. var has function scope, const has block scope.
3. let can be reassigned, const cannot.
4. var: use in cases where you need function-wide variables
let: use in most "normal" cases
const: use when a constant is needed
